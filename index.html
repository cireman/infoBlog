Essential Concepts of Web Development

What is HTML?

HyperText Markup Language. It is the most basic component of the web. It defines the meaning and structure of content.

Tags

HTML tags will help us to provide structure and semantics to the content of our website and each one has different characteristics and uses although visually they give a similar result.

Many tags are easy to differentiate such as headings or images and other different ones such as header, footer, section or div, this is where semantics will be the answer and if we know a little English we can also get a small idea of its use but in this course we will use the most important and that will be part of your day to day as a Software Developer.

DOM

The Document Object Model is a tree structure that will represent all our web projects as if it were a family tree: parents, children, siblings with infinite levels. When we understand this nesting we can identify dependencies, inheritance in css and how complex our project is.

Semantics

Semantics gives meaning to every element that exists on a website, some elements will be merely decorative and should not mean anything but others will be titles and should be in different hierarchies or some sections will be more relevant than others, this will help search engines like Google or Duck Duck Go to differentiate your content, categorize and will be the most valuable tools to be in the first SEO search results. It will also help the accessibility of your website, so that people with different abilities can understand each content.

Atributes

Attributes give extra features to the tags to complement information and are preprogrammed by the browser, some attributes work on all tags in the same way like 'class' but other attributes work on only some tags like src.

CSS

Cascading Style Sheets or CSS is the style language used to describe the presentation of HTML documents.

Box Model

The box model is composed of:

  1. Content size (width and height)
  2. padding
  3. border
  4. margin

Physical properties for the box model:

  • width / height
  • margin / margin-top | margin-right | margin-bottom | margin-left
  • border / border-top | border-right | border-bottom | border-left
  • padding / padding-top | padding-right | padding-bottom | padding-left

Logical properties for the box model:

  • inline-size / block-size
  • margin-block | margin-inline | margin-block-start | margin-block-end | margin-inline-start | margin-inline-end
  • border-block | border-inline | border-block-start | border-block-end | border-inline-start | border-inline-end
  • padding-block | padding-inline | padding-block-start | padding-block-end | padding-inline-start | padding-inline-end
